# Pribadi3 QRIS Transaction Predictive Dashboard

An end-to-end business intelligence project that combines data analytics, machine learning, and interactive visualization to analyze and predict QRIS transaction performance.

## Project Overview

This project was developed to help my family analyze their small business, identify key performance indicators, and forecast future transaction values using supervised machine learning.

The workflow covers the complete analytics lifecycle, from data preparation to predictive modeling and dashboard development.

## Business Objectives

- Monitor QRIS transaction performance over time
- Identify trends, busy hours, and seasonal patterns
- Analyze transaction behavior across different dimensions
- Predict future transaction values to support business decision-making

## Project Architecture

```text
Raw Data From Database → Data Cleaning → Data Transformation → Feature Engineering
         → Machine Learning Model → Power BI Dashboard
```
